Le jeux de l'escargot

Cet exercice nous demande d'utiliser nos connaissances pour re créer "from scratch" ( sans framework) le jeu de l'escargot. Les langages nécessaires pour cet exercice étaient en grande partie du JavaScript et CSS. Il est donc le premier exercice ou l’on utilise des fonctions de JavaScript et des algorithmes tels que le “quicksort”. Le principe du jeu était de faire un lancer de dés et de bouger les escargots de la couleur correspondante au dés quand l’escargot passe la ligne d'arrivée le joueur qui a misé sur cette couleur gagne. L'image qui suit est une image de mon rendu du jeu qui est fonctionnel.

Site Sécurisé

Site Sécurisé le projet 3 est différent de tous les autres car ce projet utilise la fonctionnalité de serveur et de base de données. C’est à dire que jusque là nos site était accessible seulement par les dossier locaux sur la même machine, mais avec ce projet de Site Sécurisé on a créé une base de donne, une page de connexion avec un mot de passe et le nom de l’utilisateur. Le fonctionnement est le suivant:

L'utilisateur rentre dans notre site web il lui est demander de rentrer c’est identifiant s' il na pas déjà de compte créer.

Voici l’exemple d’un nouvel utilisateur il crée son compte sur le site register a ce moment c’est donner sont envoyer dans la base de donné puis au moment où l'utilisateur veut se connecter le site vérifie la base de donne et si l’utilisateur a les bonne information il lui permet l'accès au site web principal .

Un générateur de citation aléatoire

Voiçi le premier Brief après la période de Prairie, dans cet exercice nous devons recréer à partir d’une base de données de citations, un programme qui affiche des citations aléatoires tout en gardant l’auteur de la citation actuelle correctement associer .Les couleur quand à elle devait aussi être aléatoirement appliquer pour chaque citation afficher . Tout cela donc a l'appui de l’utilisateur sur le bouton “new Quote” (Nouvelle citation).

Dans cet exercice nous utilisons donc les langages JavaScript, HTML et CSS le but étant de s’exercer sur le DOM (Document Object Model) afin d’appliquer une ou plusieurs fonctions à une page web et rendre nos sites Web interactives.
